2. 网页版 vs API — 别搞混

💬

网页版（免费/订阅）

ChatGPT(chatgpt.com)、豆包、通义千问。浏览器里手动打字聊天。不能集成到你的代码里。

⚡

API（按量计费）

在代码里调用AI。可以批量处理、做网站、做APP、自动化。充¥10够学一个月。

3. 国内外生态

🇺🇸

海外

OpenAI(GPT)、Anthropic(Claude)、Google(Gemini)、xAI(Grok)。需要外币付款，部分API国内直连不稳定。

🇨🇳

国内

DeepSeek、通义千问、文心一言、Kimi、豆包。人民币充值、中文能力优秀、连接稳定。

4. 开源模型 & Ollama — 跑在你自己的电脑上

如果你有好的显卡：可以用Ollama（ollama.com）在本地跑开源模型。一行命令：ollama run qwen3:14b 就能在本地跑通义千问3。好处：完全免费、数据不出你的电脑。坏处：需要好显卡（至少8GB显存），效果不如Cloud API。

5. 选型建议

🎯新手最优策略：主模型=DeepSeek V4（中文最好、最便宜、连接最稳）。复杂代码=Claude Sonnet 4.6。两个加起来月费<¥50。

不要同时付费多个模型。先用最便宜的DeepSeek，遇到它搞不定的复杂推理再开Claude。能省就省。
