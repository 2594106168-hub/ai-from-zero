🐍 Python 编程入门

AI开发90%用Python。你不需要精通，但需要看得懂、改得动、能让Claude Code帮你写。

1. 安装 Python

  1. python.org → 下载最新版 → 安装时务必勾选"Add Python to PATH"
  2. 终端输入 python --version，看到版本号=成功
  3. pip install openai requests pandas 把最常用的三个包装上

2. 变量 — 给数据贴标签

你有一个数字25，想存起来反复用。给它取个名字age。这就是变量。Python是动态类型——你不需要声明类型，解释器自己判断。

Python 
age = 25                          # int 整数
price = 99.90                    # float 浮点数
name = "张三"                    # str 字符串（单引号双引号都行）
is_admin = True                 # bool 布尔值（首字母大写！）
nothing = None                  # NoneType 空值

# f-string — 把变量插进文字里，Python 3.6+支持
print(f"{name}今年{age}岁，余额{price}元")  # 张三今年25岁，余额99.9元

3. 容器：列表/元组/字典/集合

Python 
# 【list 列表】有序、可改、可重复 — 最常用
fruits = ["🍎", "🍌", "🍊"]
print(fruits[0])          # 🍎 索引从0开始
print(fruits[-1])         # 🍊 -1=倒数第一个
fruits.append("🍇")       # 加一个到末尾
fruits.remove("🍌")       # 删掉指定元素

# 【tuple 元组】有序、不可改 — 坐标/配置项用这个
point = (3, 4)           # 不能 point[0]=5 会报错

# 【dict 字典】键值对 — 最核心的数据结构
user = {"name":"张三", "age":25, "skills":["Python","React"]}
print(user["name"])       # 张三
user["email"] = "z@qq.com" # 随时加新字段

# 【set 集合】无序、不重复 — 去重、成员检查
tags = {"AI","Python","AI"}  # 自动去重 → {"AI","Python"}

4. 条件判断

Python

5. 循环

Python 
# for 遍历每个元素
for f in fruits: print(f)

# range() 生成0到n-1的整数 — 超常用
for i in range(5): print(i)  # 0 1 2 3 4

# enumerate 同时拿索引和值
for i, f in enumerate(fruits): print(f"#{i}: {f}")

# 列表推导式 — 一行生成新列表（Pythonic写法）
squares = [x**2 for x in range(10)]  # [0,1,4,9,16,...81]

6. 函数

Python 
def calculate_price(base, discount=0, tax=0.13):
    """计算含税价格。discount=折扣率(0~1), tax=税率"""
    return base * (1 - discount) * (1 + tax)

print(calculate_price(100, discount=0.2))  # 100×0.8×1.13 = 90.4

def sum_all(*nums): return sum(nums)  # *args 可变参数
print(sum_all(1,2,3,4,5))  # 15

7. 类

Python 
class Student:
    def __init__(self, name, score):  # __init__是构造函数，创建时自动调用
        self.name = name; self.score = score
    def pass_or_not(self):
        return "✅" if self.score >= 60 else "❌"
    def change_score(self, new): self.score = new

s1 = Student("张三", 85); s2 = Student("李四", 45)
print(s1.pass_or_not(), s2.pass_or_not())  # ✅ ❌

9. 网页爬虫 — ¥100-500 的单子

Python — 爬虫（能卖钱） 
import requests; from bs4 import BeautifulSoup; import csv
url = "https://books.toscrape.com"  # 练习用，随便爬不违法
soup = BeautifulSoup(requests.get(url).text, "html.parser")
books = []; for i in soup.select("article.product_pod"):
    books.append({"title":i.h3.a.attrs["title"], "price":i.select_one(".price_color").text})
with open("books.csv","w",newline="",encoding="utf-8") as f:
    w = csv.DictWriter(f, fieldnames=["title","price"]); w.writeheader(); w.writerows(books)
⚠️爬虫有法律边界。只爬公开数据、不绕登录、尊重 robots.txt、不把数据转卖。不要爬政府网站和个人信息。
