2. OpenClaw 深入

你已装过OpenClaw。它是什么——多模型聚合网关。不绑定单一AI服务商。深度集成50+AI供应商（DeepSeek/OpenAI/Anthropic/通义千问/豆包/Ollama本地模型）。

提供统一的聊天界面+Gateway+子Agent系统+多频道接入。装完后你可以在一个界面里同时跟DeepSeek和Claude对话。支持MCP协议。支持微信/Telegram/Discord等多频道。

3. Codex CLI — OpenAI 的 Claude Code

终端 
npm install -g @openai/codex
set OPENAI_API_KEY=sk-你的Key    # Windows
codex                              # 启动

和 Claude Code 几乎一样的用法——终端里启动，说人话让它写代码。区别是底层模型是 GPT-4.1 而不是 Claude。

4. Cursor — 最强 AI 编辑器

Cursor是基于VS Code的AI编辑器。核心功能：Ctrl+K 选中代码直接说改成什么；Ctrl+L 打开侧边AI对话面板；Tab 接受AI补全。它知道你的整个项目上下文，补全准确度远高于Copilot。

下载：cursor.com。Pro $20/月。新手最推荐的日常编程工具。

5. v0.dev & Bolt.new — 零代码出页面

v0.dev：Vercel出品。你在网页上描述要什么界面，它生成 React+Tailwind 的前端页面。一键复制代码。适合快速原型、UI探索。Bolt.new：更进一步——在浏览器里直接生成+运行全栈应用（前端+后端+数据库）。MVP原型利器。

6. 新手最佳组合

🎯Cursor（日常编码）+ Claude Code（大项目）+ v0（原型）+ CC Switch（省钱）。四件套加起来月费约¥200。一人开发效率顶3个初级程序员。
