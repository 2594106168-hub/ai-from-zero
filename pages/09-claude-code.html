🦾 Claude Code

终端里的AI程序员。能读你的文件、写代码、改代码、运行命令、部署项目。不是聊天——是指挥一个虚拟程序员干活。

1. Claude Code 是什么

Claude Code 是 Anthropic 发布的命令行 AI 编程助手。它在你的终端里运行，能读你的整个项目目录、创建新文件、修改现有代码、执行终端命令、安装依赖包、部署上线。你只需要告诉它"我想做什么"，它自己做。

跟 ChatGPT 网页版的区别：ChatGPT 你复制粘贴代码自己跑。Claude Code 直接帮你改了文件还帮你跑了。

2. 安装（3步）

  1. node -v → 没装就去 nodejs.org 下 LTS 版
  2. npm install -g @anthropic-ai/claude-code
  3. claude login → 浏览器自动弹出授权

3. 核心命令

命令做什么
claude启动对话
/init初始化项目，生成 CLAUDE.md
/compact压缩对话历史省 Token
/clear清空对话
/config换模型、改配置
/doctor自检诊断

4. 用 CC Switch 走 DeepSeek（省90%成本）

Claude Code 默认用 Anthropic 的 API，较贵。你已装了 CC Switch，把后端切到 DeepSeek：

Base URL:  https://api.deepseek.com/anthropic
API Key:   sk-你的DeepSeek Key
主模型:   deepseek-v4-pro[1m]
快模型:   deepseek-v4-flash
💰走 DeepSeek 成本降低约90%。写一天代码可能只要几块钱。Anthropic 官方要几十块。

5. 实战：一句话建网站

打开终端，进入空文件夹，输入 claude，直接说：

"帮我做一个人博客网站。HTML+CSS+JS，深色主题，首页+文章列表+关于我。
先告诉我你打算怎么设计，我确认了你再写代码。"

Claude Code会：①先给你设计方案 ②你点头后自动创建文件写代码 ③你继续提修改意见 ④它直接改文件。

6. CLAUDE.md — 项目的记忆文件

项目根目录放一个 CLAUDE.md，用中文写清楚技术栈、代码风格、当前任务。Claude Code 每次启动自动读。省得每次都重新解释。

CLAUDE.md 示例 
# 项目: 个人博客
- 技术: HTML + CSS + JS, 不用框架
- 部署: GitHub Pages
- 风格: 深色主题, 紫色为主, 简洁
- 不要引入任何第三方库
- 代码加中文注释
- 修改前先告诉我你打算怎么做

7. 子 Agent — 多个 AI 同时干活

Claude Code 支持子 Agent 并行执行。你让它做一个网站，它内部可能启动一个前端 Agent 写 HTML、一个样式 Agent 写 CSS、一个逻辑 Agent 写 JS，三个同时干然后合并。不需要你插手。这对大项目效率提升巨大。
