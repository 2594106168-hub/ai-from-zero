⚡ API 编程

第一行代码就让AI为你工作。10分钟跑通。之后所有AI应用都是这段代码的扩展。

1. 调用 DeepSeek（推荐新手起步）

  1. 注册 platform.deepseek.com → 实名 → 充值¥10
  2. 左侧"API Keys"→创建→立即复制保存（只显示一次！）
  3. 终端：pip install openai
Python — 人生第一段AI代码 
from openai import OpenAI
client = OpenAI(api_key="sk-你的Key", base_url="https://api.deepseek.com/v1")
r = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role":"user", "content":"用一句话解释什么是递归"}]
)
print(r.choices[0].message.content)  # ← 这里就是AI的回复
✅跑通这段代码 = 你人生的第一段AI程序。接下来所有的聊天机器人、自动写作工具、代码生成器，本质上都是这段代码的扩展。

2. 调用 Claude

Python 
from anthropic import Anthropic
c = Anthropic(api_key="sk-ant-")
m = c.messages.create(model="claude-sonnet-4-6", max_tokens=1000,
    system="你是中文编程老师",
    messages=[{"role":"user", "content":"解释 async/await"}])
print(m.content[0].text)

注意：Claude把System Prompt和User Messages分开了。DeepSeek/OpenAI把它们混在messages数组里用role区分。

3. 多轮对话

AI本身无状态——每次请求它"失忆"。要让AI记住上下文，你得把前面的对话全放回messages数组里。

Python — 带记忆的对话 
msgs = [{"role":"system", "content": "你是日语老师"}]
msgs.append({"role":"user", "content": "「ありがとう」是什么意思？"})
msgs.append({"role":"assistant", "content": "是'谢谢'的意思。"})
msgs.append({"role":"user", "content": "再教我3个常用单词"})
r = client.chat.completions.create(model="deepseek-chat", messages=msgs)
# AI知道前面教过"ありがとう"

4. 流式输出 — 打字机效果

Python 
stream = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role":"user", "content": "写一首编程打油诗"}],
    stream=True
)
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)

5. Function Calling — AI决定调用你的工具

告诉AI你有哪些工具可用。AI自己决定调哪个工具、传什么参数。你的代码执行工具→把结果返回给AI→AI基于结果继续对话。

Python 
tools = [{"type":"function", "function":{
    "name":"get_weather", "description":"查天气",
    "parameters":{"type":"object", "properties":{"city":{"type":"string"}}}
}}]
r = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role":"user", "content":"北京今天天气怎么样？"}],
    tools=tools
)
# AI返回的不是文字——是一个tool_call：{"name":"get_weather","arguments":{"city":"北京"}}
# 你的代码执行函数→拿到结果→塞回messages里→继续对话→AI用自然语言告诉你天气

这就是 Agent 的底层机制。后面的Agent章节你会看到这个模式被放大到极致。
